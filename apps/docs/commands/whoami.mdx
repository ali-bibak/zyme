The `zyme whoami` command displays information about the currently logged-in user, such as their username and account details, making it easy to confirm which user account is active in the CLI.

## Commands

### `zyme whoami`

This command shows the currently logged-in user:

<CodeBlock language="bash">
zyme whoami
</CodeBlock>

This is useful to ensure that you are using the correct user account, especially when working across multiple projects or environments.
