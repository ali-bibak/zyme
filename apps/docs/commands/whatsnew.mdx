The `zyme whatsnew` command shows a changelog or summary of the latest features, bug fixes, and updates available in the Zyme CLI.

## Commands

### `zyme whatsnew`

This command displays a summary of recent changes:

<CodeBlock language="bash">
zyme whatsnew
</CodeBlock>

The output includes new features, bug fixes, and other updates, helping you stay informed about the latest improvements in Zyme.
