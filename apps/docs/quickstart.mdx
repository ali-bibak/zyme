title: 'Quickstart'
---

<Steps>
  <Step title="Install">
    To get started with Zyme, you'll first need to install it globally on your system. 
    
    Run the following command in your terminal:
    ```bash
    npm install -g zyme
    ```
    Once installed, Zyme will be available as a command-line tool, and you can use it within any repository.
  </Step>
  
  <Step title="Run">
    After installing Zyme, navigate to the root of your repository and run the following command:
    ```bash
    zyme
    ```
    Zyme will quickly scan the repository and provide you with a detailed breakdown of the tools, configurations, and architecture.
  </Step>
  
  <Step title="Learn">
    Once Zyme has analyzed your repo, now run:
    ```bash
    zyme learn
    ```

  </Step>
</Steps>
